계산 내역 보기
  • CIF(USD) = 물품가 + 운송료 + 보험
  • CIF(KRW) = CIF(USD) × 환율
  • 관세 = CIF(KRW) × 관세율
  • 부가세 = (CIF + 관세 + 기타세액) × 부가세율

사용방법 및 안내

계산 흐름과 자동 적용 기준을 실무 관점에서 정리했습니다.

빠른 시작
  1. 원산지/발송국가를 선택합니다. 미국은 USD 200, 그 외 국가는 USD 150 기준입니다.
  2. USD 과세환율은 첨부된 공공데이터포털 서비스키로 관세청 API에서 자동 조회하며, 필요 시 직접 수정할 수 있습니다.
  3. 물품가격·운송료·보험료를 USD 기준으로 입력합니다.
  4. HS CODE 검색에서 대표 코드를 선택하거나 관세율을 직접 입력합니다.
  5. 우측 결과 카드에서 관세·부가세·총 세액·도착원가를 확인합니다.
관세청 USD 과세환율 자동조회
  • 첨부된 공공데이터포털 서비스키를 적용해 관세청_관세환율정보(GW) API에서 USD 수입 과세환율을 자동 조회합니다.
  • 브라우저 보안 정책, API 호출 제한 또는 네트워크 제한으로 자동조회가 실패하면 유니패스 주간환율에서 확인한 값을 직접 입력하세요.
HS CODE 검색 & 관세율
  • 검색은 숫자 기준으로 대표 코드가 제안됩니다.
  • 실제 신고 전에는 관세법령정보포털, 관세사 또는 세관 품목분류를 통해 정확한 HS CODE를 확인해야 합니다.
  • FTA·협정세율·탄력세율 등은 별도 검토가 필요합니다.
보험료 자동계산
  • 자동계산 ON: 물품가격 × 110% × 0.03%, 최소 USD 20 적용.
  • 보험사, 화물 특성, 운송 조건에 따라 실제 보험료는 달라질 수 있습니다.
면세 기준
  • 미국은 CIF USD 200 이하, 그 외 국가는 CIF USD 150 이하 기준으로 면세 추정 표시합니다.
  • 목록통관 배제 품목, 과세대상 품목, 합산과세 등 예외가 있을 수 있습니다.
